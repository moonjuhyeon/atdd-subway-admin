## 지하철 역(station)

- 지하철 역 속성:
    - 이름(name)

## 지하철 구간(section)

- 지하철 (상행 방향)역과 (하행 방향)역 사이의 연결 정보
- 지하철 구간 속성:
    - 길이(distance)

## 지하철 노선(line)

- 지하철 구간의 모음으로 구간에 포함된 지하철 역의 연결 정보
- 지하철 노선 속성:
    - 노선 이름(name)
    - 노선 색(color)
